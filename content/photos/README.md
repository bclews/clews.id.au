# Photos Directory

Simply add your photo files to this directory and they will automatically appear on the photos page!

## Supported Image Formats
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)

## How It Works
The photos page automatically discovers and displays all images in this directory. No need to edit any code!

## Recommended Image Specifications
- Format: JPEG for photos
- Size: Any size works - Hugo will automatically create optimized thumbnails
- For best results: 1000x1000px to 3000x3000px

## Tips
- Images are displayed in alphabetical order by filename
- Consider naming your files with numbers or dates for ordering (e.g., `001-photo.jpg`, `2024-11-16-sunset.jpg`)
- Hugo automatically optimizes images for web performance
