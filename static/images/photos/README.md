# Photos Directory

Place your photo files here. The photos page expects images named:
- photo1.jpg
- photo2.jpg
- photo3.jpg
- photo4.jpg
- photo5.jpg
- photo6.jpg

You can add more photos by editing the `/layouts/photos/single.html` file and adding additional photo-item divs.

## Recommended Image Specifications
- Format: JPEG or PNG
- Aspect ratio: 1:1 (square) works best for the grid layout
- Size: 1000x1000px to 2000x2000px for good quality
- Optimize images for web to reduce file size

## Adding New Photos
1. Add image files to this directory
2. Edit `/layouts/photos/single.html`
3. Add new photo-item divs with the correct path and alt text
